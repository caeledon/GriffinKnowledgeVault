# 🛡 Regulators – Надзорные органы

У цьому розділі зберігаються індивідуальні профілі регуляторів:
- центральні банки;
- національні фінансові служби;
- агентства з нагляду за платіжними інститутами.

Профілі включають:
- офіційні назви;
- функції;
- підзвітність;
- сайт та джерела;
- пов’язані зони дії.
